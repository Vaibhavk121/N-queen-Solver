# N-Queen Solver

## Description

The N-Queen Solver is a web application that solves the classic N-Queen problem. The problem involves placing N queens on an NÃ—N chessboard so that no two queens threaten each other. The solver uses a backtracking algorithm to find a solution and displays it in a web-based format.

## Features

- Input the size of the chessboard (N).
- Solve the N-Queen problem and display the solution on the chessboard.
- Visual representation of the solution with queens displayed queen image.
